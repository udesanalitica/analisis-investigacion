## Descripción del repositorio

Este repositorio contiene los insumos, bases de datos, scripts, tablas, imágenes y salidas analíticas generadas en el marco del trabajo de grado titulado **“Análisis de la Investigación en la Universidad de Santander a Partir de las Capacidades de los Grupos de Investigación, la Participación Estudiantil en Semilleros y el Desempeño Académico”**.

El propósito del repositorio es organizar de manera trazable y reproducible los archivos empleados para el procesamiento, análisis y presentación de resultados relacionados con las capacidades institucionales de investigación, los grupos de investigación, la participación estudiantil en semilleros y su relación con indicadores de desempeño académico.

---

## Estructura del repositorio

```text
analisis-investigacion/
├── analisis_salidas/
├── bd_grupos_acm/
├── bd_semilleros_mlj/
├── img/
├── r_scripts/
├── tablas_scraping/
├── README.md
├── .gitignore
└── analisis-investigacion.Rproj
```

## Descripción de carpetas

* `analisis_salidas/`: contiene productos derivados del análisis, tales como resultados, reportes, tablas finales o salidas intermedias.
* `bd_grupos_acm/`: almacena las bases de datos relacionadas con grupos de investigación y capacidades institucionales.
* `bd_semilleros_mlj/`: contiene las bases de datos asociadas con semilleros de investigación y participación estudiantil.
* `img/`: reúne imágenes, gráficos, capturas o recursos visuales utilizados en los documentos del proyecto.
* `r_scripts/`: contiene los scripts desarrollados en R para limpieza, transformación, análisis estadístico y generación de resultados.
* `tablas_scraping/`: almacena tablas obtenidas, estructuradas o procesadas mediante procesos de consulta y extracción de información.

---

## Reproducibilidad

Los análisis pueden reproducirse a partir de los scripts ubicados en la carpeta `r_scripts/`. Se recomienda abrir el archivo `analisis-investigacion.Rproj` en RStudio y ejecutar los scripts de forma ordenada, verificando previamente la ubicación de las bases de datos y las rutas internas utilizadas en cada archivo.

---

## Requisitos generales

Para trabajar con este repositorio se recomienda contar con:

* R
* RStudio
* Git
* Cuenta activa en GitHub
* Paquetes de R declarados en los scripts del proyecto

Los paquetes requeridos deberán instalarse de acuerdo con las librerías especificadas al inicio de cada script.

---

## Uso del repositorio

1. Clonar o descargar el repositorio.
2. Abrir el archivo `analisis-investigacion.Rproj` en RStudio.
3. Revisar las carpetas que contienen las bases de datos.
4. Ejecutar los scripts disponibles en `r_scripts/`.
5. Consultar las salidas generadas en `analisis_salidas/`.
